Docs 소개

이 섹션은 HTML 소스만으로 문서를 추가할 수 있도록 설계되었습니다. extra/docs/ 폴더에 HTML 파일을 두면 사이드바에 자동으로 노출됩니다. 탭과 Recent에도 문서의 title이 그대로 사용됩니다.

문서 추가 방법

  1. extra/docs/template.html 파일을 복사합니다.
  2. 파일명을 적절한 슬러그로 변경합니다. 예: my-guide.html
  3. <title> 요소에 사이드바와 탭에 표시할 문서명을 입력합니다.
  4. 본문을 자유롭게 HTML로 작성합니다.

스타일

문서의 본문은 앱의 다크 테마에 맞춰 .docs-content 스타일로 감싸져 렌더링됩니다. 색상, 타이포그래피, 코드블록, 표, 인용문 등 기본 요소가 자연스럽게 보이도록 미리 정의되어 있습니다.
