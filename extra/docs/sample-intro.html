Docs 소개 및 사용 가이드

HunyDev Docs는 순수 HTML 파일만으로 기술 문서를 작성하고 공유할 수 있는 시스템입니다. extra/docs/ 폴더에 HTML 파일을 추가하면 사이드바에 자동으로 등록되며, 앱의 다크 테마와 타이포그래피가 자동으로 적용됩니다.

핵심 철학: CSS를 작성할 필요 없이 시맨틱 HTML 요소만으로 아름답고 읽기 쉬운 문서를 작성합니다.

주요 특징

  • 자동 스타일링: 다크 테마, 타이포그래피, 코드 하이라이팅 등이 자동 적용됩니다.
  • TOC 생성: h2, h3 제목이 자동으로 우측 사이드바 목차에 표시됩니다.
  • 앵커 링크: 모든 제목에 호버하면 ¶ 앵커 링크가 나타납니다.
  • 코드 복사: 모든 <pre> 블록에 자동으로 복사 버튼이 추가됩니다.
  • 원문 보기: 우측 상단 버튼으로 원본 HTML 소스를 확인할 수 있습니다.

빠른 시작

1단계: 템플릿 복사

템플릿 파일을 복사하여 새 문서를 만듭니다:

cp extra/docs/templates/template.html extra/docs/my-guide.html

2단계: 메타데이터 수정

HTML 파일의 <title> 태그를 수정합니다. 이 제목이 사이드바와 탭에 표시됩니다:

<title>나만의 가이드 문서</title>

3단계: 본문 작성

<body> 안에 시맨틱 HTML로 콘텐츠를 작성합니다. 사용 가능한 모든 HTML 요소는 Cheatsheet를 참고하세요.

4단계: R2 업로드 (선택)

로컬 파일은 자동으로 표시되며, 원하는 경우 R2 스토리지에 업로드하여 r2/ 경로로 접근할 수 있습니다.

지원되는 HTML 요소

다음 요소들이 앱의 스타일 시스템에 통합되어 있습니다:

요소 용도 스타일 특징
<h1> ~ <h6> 제목 계층 흰색, 앵커 링크, h2는 하단 보더
<p> 본문 단락 회색 텍스트, 1.75 줄간격
<strong> 강조 굵은 글씨
<em> 기울임 강조 이탤릭
<code> 인라인 코드 배경색, 패딩, 둥근 모서리
<pre><code> 코드 블록 어두운 배경, 자동 복사 버튼
<a> 링크 파란색, 밑줄
<ul> <ol> 리스트 들여쓰기, 마커 표시
<blockquote> 인용문 좌측 파란 보더, 회색 텍스트
<table> 표 보더, 헤더 배경

작성 팁

제목 구조화

h2와 h3를 사용하여 문서 구조를 명확히 하세요. 이들은 자동으로 TOC에 표시됩니다:

<h2>메인 섹션</h2>
<h3>서브 섹션</h3>

코드 블록

언어를 명시할 필요 없이 <pre><code>로 감싸면 됩니다. 복사 버튼이 자동 생성됩니다:

function example() {
  return "Hello, World!";
}

링크 참조

외부 링크는 target="_blank"와 rel="noopener"를 권장합니다:

<a href="https://example.com" target="_blank" rel="noopener">외부 링크</a>

고급 활용

중첩 리스트

리스트는 중첩하여 계층 구조를 표현할 수 있습니다:

  • 상위 항목
    • 하위 항목 1
    • 하위 항목 2
  • 또 다른 상위 항목

복잡한 표

표는 기술 문서에서 정보를 정리하는 강력한 도구입니다:

메서드 설명 반환값
get() 데이터를 가져옵니다 Promise<T>
set() 데이터를 저장합니다 void
delete() 데이터를 삭제합니다 boolean
